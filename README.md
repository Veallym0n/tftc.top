# 🗺️ TFTC Map — 中国 Geocaching 地图工具

> **Thanks For The Cache！** 专为中国 Geocaching（地理藏宝）玩家打造的在线地图工具。
>
> 🌐 **立即访问：[tftc.top](https://tftc.top)**

---

## 这是什么？

[tftc.top](https://tftc.top) 是一个运行在浏览器里的 Geocaching 辅助地图，**不需要安装任何 App，打开网页就能用**。

Geocaching 官方 App 和网站在国内有一个长期存在的痛点：地图底图是 Google Maps，在中国大陆网络环境下加载缓慢甚至无法使用，而且坐标存在偏移，导致图钉落点不准确。**TFTC Map 就是为了解决这些问题而生的**——它使用国内可以正常访问的高德地图作为底图，并自动修正坐标偏差，让你在国内也能流畅、准确地浏览藏点地图。

---

## 解决了哪些问题？

### 🌏 国内地图访问问题
官方工具依赖 Google 地图，在国内加载极慢或直接打不开。TFTC Map 默认使用**高德地图**底图，在国内网络环境下秒开，也支持切换到 OpenStreetMap。

### 📍 坐标偏移问题
中国大陆强制要求地图使用 GCJ-02 坐标系（俗称"火星坐标"），而 Geocaching 藏点数据使用的是国际通用的 WGS-84 坐标。如果不做转换，图钉会偏离真实位置几十到几百米。TFTC Map 会**自动识别并转换坐标**，确保图钉落在正确位置。

### 🐌 加载慢、筛选卡顿问题
藏点数量多时，每次切换筛选条件都要重新请求网络，体验很差。TFTC Map 在第一次同步后将所有数据**缓存在本地**，之后的筛选全部在本地瞬间完成，切换 Today / FTF / Event 等模式几乎无延迟。同时图钉采用**聚合算法**，就算有几千个藏点同时显示也不会卡顿。

### 📵 野外无信号问题
寻宝时经常进入信号差的地方。数据本地化之后，**断网依然可以正常浏览地图和藏点信息**，不会因为没有信号就一片空白。

## 主要功能详解

### 🗺️ 地图浏览
在地图上查看藏点分布，点击图钉即可查看藏点名称、编号、类型、难度、地形等基本信息。支持长按地图快速获取该位置的坐标。

### 🔍 多维度筛选
根据不同的寻宝需求，快速切换不同的藏点视图：
- **Today**：今天有活动或新发布的藏点
- **Event**：附近的 Event 活动藏点
- **Found**：你已经找到的藏点
- **FTF**：有机会首找（First To Find）的藏点
- **全量**：显示所有已同步的藏点

### 📡 离线数据缓存
完成一次数据同步后，藏点数据会存储在你的设备上。之后打开工具无需联网，所有藏点和筛选功能照常使用。出发前在 WiFi 环境同步一次，野外放心用。

### 🧭 坐标工具
- 输入任意坐标，地图立刻跳转到对应位置
- 支持 Geocaching 常用的"度分"格式（如 `N 31° 13.800 E 121° 28.400`）自动解析
- 坐标在 WGS-84 和 GCJ-02 之间自由转换，方便与其他导航工具互通

### 💾 GPX 导出
将当前筛选的藏点导出为 GPX 文件，可以直接导入 Garmin 等手持 GPS 设备，或者导入 OsmAnd、Maps.me 等手机导航 App 离线使用。

### 🔗 深链接跳转
支持通过 URL 参数直接定位到某个藏点或坐标位置，方便在寻宝群里分享精确位置。

## 它是怎么工作的？

整个工具运行在你的浏览器中，不需要注册账号，也没有独立的服务器存储你的数据：

1. **同步数据**：通过 Geocaching 平台的接口拉取你的藏点列表，一次性存入浏览器本地存储，后续无需重复请求
2. **本地过滤**：所有筛选条件（今日、FTF、Event 等）都在本地数据库中完成，切换瞬间响应
3. **地图叠加**：以高德或 OSM 为底图，将本地数据渲染为图钉，超过一定数量自动聚合成气泡，避免卡顿
4. **坐标纠偏**：所有坐标在显示时自动完成 WGS-84 → GCJ-02 转换，图钉位置与实际地貌吻合

## 开始使用

无需注册，无需安装，直接访问 👉 **[tftc.top](https://tftc.top)**

首次使用步骤：
1. 打开网站，点击左侧菜单栏
2. 在「数据集」中选择数据来源，点击同步
3. 同步完成后地图上即可看到藏点，之后断网也能正常浏览
